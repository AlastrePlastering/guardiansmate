# GuardiansMate Website

Static website ready for GitHub and Vercel.

## Folders

```
index.html
src/styles.css
src/main.js
public/assets/
robots.txt
sitemap.xml
vercel.json
```

## Important

Place your real PNG images inside `public/assets/` using these exact names:

- brand_badge.png
- hero_charms.png
- charms_grid.png
- straw_charms.png
- necklaces.png
- owl.png
- panda.png
- cloud.png
- capybara.png
- dog.png

Update the Instagram and TikTok links inside `index.html` when your final handles are ready.

For production SEO, replace `https://guardiansmate.com/` in `index.html`, `robots.txt`, and `sitemap.xml` with your final domain if different.
